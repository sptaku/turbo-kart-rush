# 🏁 TURBO KART RUSH（ターボ・カート・ラッシュ）

マリオカート風の**擬似3D（Mode 7 方式）**レースゲーム。自分のカートの後方視点で、地面が遠近で奥に広がる“スーパーマリオカート”のような見え方です。HTML5 Canvas + Web Audio 製で、追加ライブラリ不要・**ブラウザで開くだけ**で遊べます。BGM・効果音はすべて Web Audio による**オリジナル生成**（著作権物は不使用）です。

## あそびかた

`index.html` をダブルクリックしてブラウザで開くだけ。
（音が出ない場合は最初に画面を1回クリック／キーを押すと有効化されます。ブラウザの自動再生制限のため。）

ローカルサーバ経由でも動きます:
```
cd mario_cart_20260615
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 操作

| | 1P（レッド） | 2P（ブルー） |
|---|---|---|
| アクセル | **W** | **↑** |
| ブレーキ / バック | **S** | **↓** |
| ハンドル | **A / D** | **← / →** |
| ドリフト（溜めてミニターボ） | **左Shift** | **右Shift** |
| アイテム使用 | **Space** | **Enter** |
| **シフトアップ**（ギアを上げる） | **E** | **.（ピリオド）** |
| **シフトダウン**（ギアを下げる） | **Q** | **,（カンマ）** |
| ポーズ | **Esc** | |

> ※ 1P は左手側キー・2P は右手側キーに分けています。キーボードによっては多数同時押しを取りこぼす（ゴースティング）ことがあります。

> **3〜8人のキーボード割当**（ハンドル左/右・アクセル・ブレーキ・ドリフト・アイテム・ギア▲▼）: 3P=**I J K L**系（U/O/P/;）、4P=**T F G H**系（R/Y/B/V）、5P=**テンキー**、6P=**数字段**、7P=**Z X C N**系、8P=**記号キー**（詳しくは `js/input.js`）。**5〜8人は1台のキーボードだと密集するためコントローラ推奨。** Bluetooth/USB ゲームパッドは接続順にプレイヤー番号へ自動割当（**iPhone / iPad のブラウザ**でも使えます）。

### 📱 スマホ（タッチ操作）

タッチ端末（**iPhone / iPad** など）で開くと、レース中だけ**半透明の操作パネル**が自動表示されます（左下＝◀▶ハンドル、右下＝アクセル／ブレーキ／アイテム／ドリフト）。**セミAT・MT を選ぶと「ギア▲／ギア▼」ボタン**も表示され、タッチでも変速できます（ATは自動変速なので不要）。スピードメーターは親指に重ならないよう画面下中央へ移動します。**横向き**でのプレイがおすすめです。
（iPad はデスクトップ表示の Safari でも `maxTouchPoints` で判定するため iPhone と同じく操作できます。）

### ミッション（変速）と ギアチェンジ操作

タイトル画面の **「⚙ ミッション」** ボタンで オートマ（AT）→ セミオートマ → マニュアル（MT）を切り替えます。

- **オートマ（AT）** … ギアは速度に応じて全自動。シフト操作は不要。
- **セミオートマ** … 基本は自動変速だが、**シフトアップ / シフトダウン**キーで任意に変速も可能。
- **マニュアル（MT）** … 変速は完全に手動。低ギアは加速重視・高ギアは最高速重視で、コーナー手前のシフトダウンが速さのカギ。

画面下のメーターに現在のギア（**N / R / 1〜5**）と変速モード（**AT / セミAT / MT**）が表示されます。

## 要件と実装

- **擬似3D（後方視点）** … 自分のカートの後ろから見る Mode 7 風の見え方。1人プレイは全画面＋操作アシスト（曲がりやすく・スピンも短い）に特化。
- **立体的なカート描画** … 車体は縦グラデ＋スペキュラ反射、タイヤは円柱シェーディング、ヘルメットは球状ハイライト、接地影とリムライト付きで、平面でなく立体に見えるように描画。
- **分割画面の最大8人対戦** … VSは通常 **1〜4人**（2人＝上下分割、3〜4人＝2×2分割）。**VSを8回遊ぶと「1〜8人」が解放**され、5〜6人＝3×2分割、7〜8人＝4×2分割で各プレイヤーの後方視点を別々に描画します（解放は `localStorage` 保存。余ったセルには順位表を表示）。**5〜8人はコントローラ推奨**（1人のキーボードで8人分のキーは密集するため。Bluetooth/USBゲームパッドはプレイヤー番号順に割り当て）。
- **グランプリの中断・再開** … グランプリは各戦終了時に進捗を `localStorage` に保存。途中でやめてブラウザを閉じても、次回タイトルに**「▶ グランプリの続きから（第N/6戦）」**が出て、成績・設定（CPU台数/強さ/変速/ミラー等）ごと**やめたところから再開**できます。最終戦を終えると保存は消去。
- **10コースから選択**（**3周でゴール**。タイムアタック / VS / 「コースを変える」で選べます。グランプリは全10コース連戦）
  - 🌱 グリーン・メドウ（超かんたん・最大の道幅で初心者向け）
  - ☀️ サンシャイン・スピードウェイ（かんたん）
  - 🏜️ デューン・チェイス（ふつう・S字＋オイルでスリップ）
  - 🌃 ネオン・サーキット（むずかしい・タイトコーナー＋氷で滑る）
  - 🌋 マグマ・ガントレット（激ムズ・入り組んだ最狭コース＋マグマのスリップ）
  - 🌌 ヴォイド・スパイラル（超激ムズ） … 全コース最狭・最薄路肩の**蛇行サーキット**に**氷**を散りばめた難コース。
  - 🌈 **スカイ・ハイ・ループ（超超激ムズ・新コース）** … 明るい大空と雲海に浮かぶ細い光の道。**全コース最多のジャンプ台**でつねに宙へ舞う**立体的**な走りが特徴。氷＋細い道＋流れるシケインで最難関。
    - **🌉 橋が途切れたジャンプ区間（このコースだけの仕掛け）** … 下ストレートの途中で**コースが大きく途切れて広い奈落（崖）**になっており、その手前に**ジャンプ台が“走路＋路肩いっぱい”に横一列**に並びます（端を通ってもすり抜け不可）。崖手前のジャンプ台は**強力に飛ばす**ので、速度を保てば飛び越えられます。飛べず（低速・スピン）に落ちると**大ダメージ＋手前に引き戻し**で再挑戦。奈落は**深さのある暗い谷＋ギザギザの破断面＋落下する道路の破片**で、はっきり「崖」に見えます。
  - 🌠 **オーロラ・スプリット（超超超激ムズ・新コース／分岐あり）** … 星空とオーロラの夜コース。**道が二手に分かれる「分岐」**が他コースに無い目玉。**外回り（本線）＝遠回りだが安全**／**内回り（分岐＝ショートカット）＝近道だが途中に広い奈落ジャンプ**。CPUは安全な外回りを走るので、プレイヤーが分岐の近道を**飛んで突破**すれば一気に差をつけられます（失敗すれば落下）。最狭級の道＋氷＋上部シケインで最高難度。
  - 🌩️ **テンペスト・ゲイル（超超超超激ムズ・新コース／突風）** … 鉛色の嵐サーキット。**直線に「突風（横風）ゾーン」**があり、乗ると**矢印の向きへカートが流される**＝逆らってハンドルを当て続けないと外へ押し出される、これまでに無いギミック。さらに下直線の**奈落ジャンプ**＋氷＋最狭級の道。
  - 🌀 **ワープ・ラビリンス（超超超超超激ムズ・新コース／ワープ＋複数分岐）** … 宇宙の迷宮。**道が複数に分岐**し、ひとつは**「ワープゲート（瞬間移動）」**——光るポータルに飛び込むと右側を丸ごと飛び越えて一気に上部へワープする、これまでに無いギミック。もうひとつは上部を内側で抜けるタイトな近道。CPUは安全な本線を走るので、プレイヤーがワープ＆近道を使いこなせば大逆転できます（外すと大きく不利）。最狭級の道＋氷で全コース最難関。
- **コースアウトの意味** … 走路を外れて芝生に乗ると大きく減速。レール外には出られません。
- **マリオカート風アイテム** … 🍄キノコ / 🍌バナナ / 🟢グリーンボール（直進）/ 🔴レッドボール（追尾）/ ⭐スター / 💣ボムへい
- **気持ちよく走れる音楽（2スタイル選択可）** … コース別の軽快なBGMを合成。タイトルの **「🎵 音楽」** で **ピコピコ（チップチューン）** と **オーケストラ（弦・金管・ピッツィカート・ティンパニ＋ホール残響）** を切替できます。すべて手続き的に合成したオリジナル。
- **無敵中だけ別BGM** … ⭐スター取得中は高速アップテンポの専用BGMに切り替わり、解除で元に戻ります。
- **6車種のカート** … F1 / ゴーカート / バギー（ロールバー＋極太タイヤ）/ スポーツ（屋根＋リアガラス）/ ワゴン（背高ボディ）/ マッスル（支柱付きスポイラー）と、色だけでなく形のちがう車種が混走します。
- **マリオカートにない独自要素** … 🪝 **グラップル・ダッシュ**
  前を走るカートに自動ロックオンし、そこへ高速で引き寄せられる追い上げ専用アクション（引き寄せ中は無敵で、ぶつかった相手を弾き飛ばす）。前方に相手がいなければコース前方へダッシュ。フック/グラップルはマリオカートには無い要素です。後方ほど出やすい逆転アイテム。

- **ジャンプ台** … コース上のランプを踏むと空中へジャンプ（着地でちょいブースト）。空中はバナナ・ボール・他カートをすり抜けます。
- **❤️ ライフ（耐久）システム** … 各カートにライフがあり、**他のカートにぶつかる／コース外（芝生）に出ると減少**。**0で爆発**します（CPUもプレイヤーも同条件）。コース上の**緑のピット（少し長め）**を通るとライフが回復（F-ZERO風）。プレイヤーのライフは画面左上に**バー表示**（残量で緑→黄→赤、危険時は点滅）、他カートも被ダメージ時に小バーを表示。
  - **CPUが爆発 → リタイヤ（コースから消える）**。順位や左上の台数表示は「残っている台数」になります。
  - **プレイヤーが爆発 → そのレースは脱落（操作不能・画面が暗転して「リタイア」表示）**。VSで**他のプレイヤーが1人でも生きていればレースは続行**し、全員が爆発するか残りがゴールした時点で終了。**全員脱落でゲームオーバー**（グランプリは「このコースを最初から」やり直し・成績は維持）。
  - **🚑 リタイヤCPUの復活設定**：タイトルのトグルで、グランプリの**次のステージでリタイヤしたCPUを復活させるか**（既定ON）／**復活させず台数が減っていくサバイバル**（OFF）を選べます。中断・再開にも保存されます。
  - **🌱 コース外ダメージ ON/OFF**：タイトルのトグルで、**コース外（芝生）に出たときにライフが減るか**を切替（既定: あり）。「なし」にすると芝生でのダメージはゼロ（減速はそのまま）。衝突によるダメージは引き続き有効です。
  - **💥 ダメージ倍率（自分／CPUを独立設定）**：タイトルに **「💥 自分のダメージ」** と **「💥 CPUのダメージ」** の2つのトグルがあり、それぞれ別々に **通常 → 1/2 → 1/3 → 1/5 → 1/10 → なし** を選べます（**既定はプレイヤー=1/2・CPU=通常**＝自分はやられにくく、敵は標準）。**CPU側はさらに「ランダム」も選択可**（各CPUにバラバラの倍率を割り当て＝硬い敵・脆い敵が混在）。ランダムは **「なし含む」**（0倍＝無敵CPUもあり得る）／**「なし除く」**（必ず多少はダメージ）を選べます。**ランダムの抽選候補は今選べる倍率に連動**し、**🌟スーパー／🔥ハイパーカオスをONにすると、追加された倍率（2倍・1.5倍・1/15・1/20／3倍・2.5倍・1/25・1/30）もランダムの候補に入ります**。「なし」は無敵（ライフが減りません）。**🌟スーパーカオスON時は追加で 2倍・1.5倍・1/15・1/20**、**🔥ハイパーカオスON時はさらに 3倍・2.5倍・1/25・1/30 も選択可能**（その段未満に戻すと自動で通常へ戻ります）。
- **🏎 最高速倍率** … タイトルのトグルで **通常 → 1.5倍 → 2倍 → 2.5倍 → 3倍** を選択。全カート（自分もCPUも）の最高速・加速・旋回を一律でスケールするので、コーナーの曲がりやすさは保ったまま**レース全体が高速化**します。
- **⚛ 高精度物理モード** … タイトルのトグルでON/OFF。ONにすると**1フレームを4分割して積分**し、衝突・壁ずり・高速時の挙動がより正確に（貫通しにくく）なります（負荷は少し増えます）。最高速倍率と相性◎。
- **📺 白黒（昭和）モード** … タイトルのトグルでON/OFF。画面全体を**モノクロ化**し、**走査線・四隅のビネット・わずかなフリッカ・同期ずれ風の走査バー**を重ねて、昭和の白黒ブラウン管TVのような見た目になります。レース中もメニューも白黒に。設定は `localStorage` 保存。
- **🪞 ミラーモード（隠し要素）** … グランプリを**全レース1位**でクリアすると解放。コースが**左右反転**し、オプションで**操作も反転**できます。解放状態は `localStorage` に保存され、次回起動時も解放済みのまま。
- **🔁 アクセル/ブレーキ反転モード（隠し要素2）** … **ミラー＋操作反転をONにしたままグランプリを完走**すると解放。ONにするとアクセルとブレーキが入れ替わる超カオスモード。こちらも `localStorage` に保存。
- **🌀 カオスモード** … タイトルのトグルでON/OFF。ONにすると**CPUの台数が設定の10倍**に（例: グランプリ9台→90台、VSのCPU3台→30台）。最大で約190台の大混戦！（重い場合は「🖼 画質: シンプル」を併用）
  - **🌟 スーパーカオスモード（隠し）** … タイトルで「🌀 カオスモード」を**5回連打**すると解放＆ON。**CPUが設定の15倍**（グランプリ最大19×15＝約285台！）。解放するとボタンが金色になり、以後は OFF→カオス→スーパーカオス を循環選択できます（解放状態は `localStorage` 保存）。
  - **🔥 ハイパーカオスモード（隠し2）** … スーパーカオス解放後、カオスボタンを**10回連打**すると解放＆ON。**CPUが設定の20倍**（グランプリ最大19×20＝約380台！）。スーパーより上。ボタンが赤くなり、循環に ハイパーカオス が加わります（解放状態は `localStorage` 保存）。
  - **💥 エラーカオスモード（隠し3）** … ハイパー解放後、カオスボタンを**13回連打**すると解放＆ON。**CPUが設定の100倍**（グランプリ最大19×100＝約1900台！）。台数が多すぎてスタートで重なる“破綻＝エラー”が起こるネタ枠。ボタンが虹色グリッチに。
  - **🌪 本当のカオスモード（隠し4・真の最高位）** … エラーカオス解放後、カオスボタンを**31回連打**すると解放＆ON。**CPUが設定の200倍**（グランプリ最大19×200＝約3800台！）。エラーより上の真の頂点。ボタンが虹色に回転します。
  - ※ 連打判定は「直前クリックから0.6秒以内なら継続」方式なので、テンポよく押し続ければ 5→10→13→31 と長い連打にも到達できます（解放状況はすべて `localStorage` 保存）。
