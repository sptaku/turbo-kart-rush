あそびかた

1P（レッド）

  • W アクセル / S ブレーキ・バック
  • A / D 左右ハンドル
  • 左Shift ドリフト（曲がりながら溜めてミニターボ）
  • Space アイテム使用
  • E シフトアップ / Q シフトダウン（セミAT・MT時）

2P（ブルー）

  • ↑ アクセル / ↓ ブレーキ・バック
  • ← / → 左右ハンドル
  • 右Shift ドリフト
  • Enter アイテム使用
  • .（ピリオド） シフトアップ / ,（カンマ） シフトダウン（セミAT・MT時）

3P（イエロー）

  • I アクセル / K ブレーキ・バック
  • J / L 左右ハンドル
  • U ドリフト / O アイテム使用
  • P シフトアップ / ;（セミコロン） シフトダウン

4P（グリーン）

  • T アクセル / G ブレーキ・バック
  • F / H 左右ハンドル
  • R ドリフト / Y アイテム使用
  • B シフトアップ / V シフトダウン

🎮 コントローラ（Bluetooth/USB）

  • 接続後、ボタンを一度押すと有効化されます
  • A / R2 アクセル / B / L2 ブレーキ
  • 左スティック / 十字キー ハンドル
  • R1 ドリフト / X アイテム
  • Y シフトアップ / L1 シフトダウン
  • P1=1台目, P2=2台目…の順に割り当て（キーボードと併用可）

VS は 1〜8人＋CPU 0〜20台。3〜4人は画面を2×2分割。3人時は右下に順位表が出ます。各プレイヤーはキーボードでもコントローラでも操作できます。

ミッション（変速）

タイトルの ⚙ ミッション で AT（全自動）→ セミAT（自動＋手動シフト可）→ MT（完全手動）を切替。MT/セミAT では上のシフトキーでギア（N/R/1〜5）を操作します。
